# CN-002 Canonical Versioning

Status: Active
Created: 2026-02-02
Confidence: verified

Rule:
All versioned artifacts in the canon MUST use semantic versioning in the form vMAJOR.MINOR.PATCH. Breaking changes require a MAJOR increment and a migration guide in evolution/migration-guides.

Scope:
Schema files, UI contracts, and any rule or decision that declares a version.
